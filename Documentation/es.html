Ayuda de IR-L-AHK

Script para usar el plugin vST Waves IR-L con lectores de pantalla.

Si bien debiece funcionar con cualquier DAW, ha sido probado con:

  • Waves IR-L 12.0
  • Reaper 6.15
  • NVDA, Jaws for Windows y Sapi5.
  • Windows10 64bits

Licencia GPL-3.0

Código fuente y últimas versiones:

https://github.com/crisspro/IR-L-AHK

Cómo usarlo

  1. Ejecuta IR-L-AHK.
  2. abre Waves IR-L desde tu secuenciador.
  3. Usa los atajos de teclado del script.
  4. Puedes cerrar el script desde la bandeja del sistema

Nota:
Si el script no funciona en la ventana FX de Reaper, usa el plugin activándolo en primer plano.

Atajos de teclado

Atajo Descripción
A Cambia al preset anterior
S Cambia al siguiente preset.
L Despliega el menú para cargar presets.
G Despliega el menú para guardar presets.
M Marca un preset
C Cambia entre el preset actual y el preset marcado
Z Deshacer los cambios
R rehacer los cambios
F1 Abre la Ayuda del script
CTRL+q Sale del script
