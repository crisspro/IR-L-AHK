IR-L-AHK help

Script to use Waves IR-L VST with screen readers.


    This script maybe be working with any DAW, was tested with:

  • Waves IR-L v12.0.
  • Reaper 6.15.
  • NVDA, Jaws for Windows and Sapi5.
  • Windows 10 x64.

Source code and last releases:

https://github.com/crisspro/IR-L-AHK

Usage

  1. Run IR-L-AHK
  2. Open Waves IR-L in your DAW.
  3. Press the hotkeys of script.
  4. Can close the script from system tray

Note:
If the script not working in Reaper's FX window, use the bringe window to Waves IR-L.

Hotkeys

Hotkey Description
S Next preset.
A Back preset.
L Open the menu to load impulse.
G Open the menu to save impulses
M Mark preset.
C Change between current preset and marked preset.
Z Undo the changes.
R Redo changes
F1 Open help documentation.
CTRL+q close the script.
